Realtime DB seed instructions (no billing required)

1. Open your Firebase console → Realtime Database → Data. Make sure you created the database.
2. Use the 'Import JSON' button to upload JSON files directly or paste the JSON into the editor.
   - Upload seed/realtime_subjects.json to /subjects
   - Upload seed/realtime_lessons.json to /lessons
   - Upload seed/realtime_pastpapers.json to /pastPapers
   - Upload seed/realtime_quizzes.json to /quizzes
3. All media use public links (YouTube embed or public WAV). No Storage upload is required.
Firebase Realtime Database URL used in this project: https://kcse-learning-hub-8a30c-default-rtdb.firebaseio.com/
